<h1>Vite + FastAPI</h1>



<h1>Запуск фронта</h1>
```bash
$ bun i 

$ bun run dev
```

<h1>Запуск бека</h1>
```bash
$ pip install -r requiremets.txt 

$ uvicorn main:app -reload
```

<h1>Docker</h1>

```bash

$ docker compose build

$ docker compose up
```

<h1>Порты</h1>

```bash

$ http://localhost:5173 #front

$ http://localhost:8000 #back

```
